## 🚀 Como Executar

### Python
```bash
python nome_do_ficheiro.py
```

### C
```bash
gcc nome_do_ficheiro.c -o output
./output
```

### Java
```bash
javac MergeSort.java
java MergeSort
```

### Dashboard Web
Abre o ficheiro `eda_dashboard.html` diretamente no browser.
